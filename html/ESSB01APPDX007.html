Appendix 5
-ing
Verbs that end in -e (make/write/drive etc.) → -eing:
make → making
write → writing
come → coming
dance → dancing
Verbs that end in -ie → -ying:
lie → lying
die → dying
tie → tying
stop → stopped, big → bigger etc.
